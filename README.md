# SHIFT---CLI-Sorter

Тестовое задание на стажировку SHIFT по направлению Java-backend от Рушковского Никиты.

## Установка
Проект требует наличие:
* Java 25
* Maven (3.9.12)

Зависимости проекта достумны в pom.xml
```bash
git clone https://github.com/Seamly71/SHIFT---CLI-Sorter.git
cd SHIFT---CLI-Sorter
# Проверяем установлен ли Maven
mvn --version
mvn clean package
# Проект собран на Java 25
java --version  
java -jar target/cli-sorter-1.0-SNAPSHOT.jar
```

## Специфика приложения
* Пути, содержащие пробелы стоит обрамлять в кавычки.
* Приложение может принимать флаги как в начале строки, так и в конце.
Если слово не является флагом или аргументом флага, то считается,
что слово обозначает путь к файлу.
* Флаг -o требует наличия директории для записи. Этот флаг не создает новые директории.
* Флаг -a требует наличия выходных файлов для записи. Этот флаг не создает новые файлы.
* Флаги -f и -s считаются взаимоисключающими.
